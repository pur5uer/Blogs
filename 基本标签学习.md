# 基本标签学习

要点在注释里

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>基本标签学习</title>
</head>
<body>

<!--标题标签-->
<h1>一级标题</h1>
<h2>二级标题</h2>

<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>

<!--段落标签-->
<p>文件File也是一个逻辑概念。计算机的所有设备都可理解为一个文件。流可与文件建立联系。</p>
<p>1、IO流类一旦被创建就会自动打开。</p>
<p>2、通过调用close方法，可以显式关闭任何一个流。</p>
<p>3、如果流对象不再被引用，java的垃圾回收机制也会隐式地关闭它。</p>

<!--水平线标签-->
<hr/>

<!--换行标签-->
文件File也是一个逻辑概念。计算机的所有设备都可理解为一个文件。流可与文件建立联系。<br/>
```
